## Berkeley Light theme

Inspired by the Berkeley colorscheme for Anvil by 非公開:
https://anvil-editor.net/themes/

Berkeley Light is a minimalist light theme shared across tmux, Alacritty, and Vim.

Core colors:
- background: white `#FFFFFF`
- foreground: black `#000000`
- muted: grey `#9A9A9A`
- green `#0A805D`
- orange `#E25600`
- gold `#FEB908`
- red `#B73A34`
- purple `#5f00b9`
- blue `#006ec3`

Design philosophy:
1. Use a low number of colors. The palette is built around background, foreground, an optional muted neutral, and 4 to 5 accent colors.
2. Maintain good contrast. The target is at least WCAG AA, `4.5:1`, for text/background combinations wherever the theme controls both colors.
